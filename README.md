# 🤖 QA Agent Task – Roshani Yadav

An AI-powered automation agent that extracts how-to knowledge from videos and documents to:
- ✅ Generate frontend test cases using GPT
- ✅ Auto-convert to Playwright test scripts
- ✅ Execute tests and report via Streamlit dashboard

---

## 🧩 Features

- 🎥 Whisper-based transcription from video
- 🧠 GPT-4o-powered test case generation (JSON + Markdown)
- 🧪 Playwright test script conversion
- 📊 Streamlit dashboard for pass/fail results, screenshots, and logs
- 🌐 CI-ready and extendable (optional)
